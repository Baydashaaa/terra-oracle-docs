Terra Oracle Classic

A decentralized Q&A, governance and draw protocol built on Terra Classic — powered by LUNC.

Terra Oracle Classic allows community members, validators, and developers to surface privileged knowledge anonymously, participate in governance draws, and engage in on-chain communication. The protocol operates entirely on Terra Classic (columbus-5) with no centralized backend. All payments, draws, and messages are verifiable on-chain.

What's included

Oracle Q&A
Ask questions about Terra Classic governance, validators and market. 200,000 LUNC per question.
Oracle Draw
Daily & Weekly on-chain draws. Mint Oracle Mask NFTs to enter. Fully verifiable.
Treasury
Protocol treasury funded by Q&A, Chat, and Draw fees. Transparent on-chain.

Oracle Q&A

Ask questions about Terra Classic governance, validators and market. Get answers from the community.

What is Oracle Q&A?

Oracle Q&A is an on-chain question and answer system. Anyone can submit a question by paying 200,000 LUNC. The payment is split automatically, and the question author receives 2 free Weekly Draw entries.

Payment split per question (200,000 LUNC):
50% → Weekly Draw Pool  ·  50% → Protocol Treasury  ·  +2 free Weekly Draw entries for author

How to ask a question

1
Connect your wallet on Terra Oracle Classic
2
Go to the Ask page
3
Select a category and write your question
4
Confirm the transaction — 200,000 LUNC
5
Your question appears on the Oracle Board

Protocol

DAO Chat

On-chain messages stored forever on Terra Classic. Every message costs 5,000 LUNC.

What is DAO Chat?

DAO Chat is a permanent on-chain messaging system. Every message is stored on Terra Classic blockchain and is publicly visible forever. The cost per message is 5,000 LUNC, which goes entirely to the Protocol Treasury.

100% of Chat fees → Protocol Treasury

Earn Weekly Draw entries

Active participation in DAO Chat is rewarded with Weekly Draw entries:

Every 10 messages sent = 1 free Weekly Draw entry
Maximum 2 entries per day can be earned through Chat

How to send a message

1
Connect your wallet on Terra Oracle Classic
2
Go to the Chat page
3
Type your message and confirm the transaction — 5,000 LUNC
4
Your message is stored on-chain permanently

Oracle Draw

Two draws — Daily & Weekly. Mint Oracle Mask NFTs to enter. On-chain verifiable draws on Terra Classic.

Oracle Draw is not a lottery — it is a mechanism designed to stimulate on-chain economic activity.

How the winner is selected

The winner is selected using the on-chain block hash at draw time — fully transparent and verifiable by anyone:

winner = block_hash % total_entries

Each NFT tier gives a different number of entries — more entries means a higher chance of winning.
